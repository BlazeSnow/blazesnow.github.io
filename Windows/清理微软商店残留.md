# 清理微软商店残留 | Windows相关

> 本篇适用于已经清空```D:\```内所有```Microsoft store程序```后，需要清理微软商店残留的用户

::: danger 警告：无法恢复
删除文件及文件夹后将无法恢复
:::


1. 按下```Windows徽标```+```X```
2. 选择```终端管理员```
3. 进入目录：```cd D:\```
4. 获取```WindowsApps```的所有权：```takeown /F "WindowsApps" /A /R /D Y```
5. 获取```WindowsApps```的管理权：```icacls "WindowsApps" /T /grant administrators:F```
6. 删除以下文件及文件夹：
   - ```DeliveryOptimization```
   - ```WindowsApps```
   - ```WpSystem```
   - ```WUDownloadCache```
   - ```XboxGames```
   - ```.GamingRoot```
