# 系统重新安装 | Windows相关

## 在本机安装

1. 跳转至Windows网站：<https://www.microsoft.com/zh-cn/software-download/windows11>
2. 选择```下载适用于 x64 设备的 Windows 11 磁盘映像 (ISO)```
3. 选择适合的版本
4. 选择适合的语言
5. 点击```立即下载```
6. 对下载的```Win11_版本号_语言_x64.iso```，解压缩到空闲位置
7. 运行其中的```setup.exe```


## 使用U盘安装

1. 跳转至Windows网站：<https://www.microsoft.com/zh-cn/software-download/windows11>
2. 选择```创建 Windows 11 安装媒体```
3. 点击```立即下载```
4. 打开下载的```MediaCreationTool.exe```
5. 插入U盘并在程序中选中U盘
6. 等待下载
7. 完成后重新启动电脑
8. 根据电脑型号按下按键进入BIOS
9. 修改第一启动项为U盘
10. 保存并退出，随后开始安装
