# 重新安装 | Windows

1. 提前查询电脑进入BIOS的按键
2. 提前准备空U盘，将U盘插入电脑
3. 打开链接：<https://www.microsoft.com/zh-cn/software-download/windows11>
4. 下载`Windows 11 媒体创建工具`并运行
5. 创建好安装媒体后，重新启动电脑
6. 在电脑启动过程中连按按键以进入BIOS
7. 在BIOS中选择UEFI启动，并选择U盘作为第一启动设备
8. 保存并退出BIOS
9. 等待电脑启动并进入Windows安装界面
