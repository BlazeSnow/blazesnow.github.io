# <TitleIcon icon="nodedotjs" /> npm管理包

> 本篇对包的安装卸载过程以`typescript`为例

## 安装包

```bash
npm install typescript
```

## 列出包

```bash
npm ls --depth=0
```

## 卸载包

```bash
npm uninstall typescript
```

## 更新所有包

```bash
npm update
```
