## 安装VitePress

1. 前往工作目录，如```C:\用户名.github.io```
2. 右键空白位置，选择```在终端中打开```
3. 建立git仓库：```git init```
4. npm换源：```npm config set registry https://registry.npmmirror.com```
5. 安装VitePress：```npm add -D vitepress```
