# 处理远程仓库 | Git相关

> 处理git仓库，需要在目标文件夹内操作

## 克隆远程仓库到本地

> 此操作需在目标文件夹上级文件夹操作，
> 将在文件夹内生成一个与远程仓库同名的文件夹

```git clone <链接地址>```

## 添加远程仓库链接

```git remote add origin <链接地址>```

## 查看远程仓库详细信息

```git remote -v```

## 删除远程仓库链接

> 仓库名在详细信息中有显示

```git remote remove <仓库名>```
