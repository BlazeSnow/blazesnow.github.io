# 安装Nodejs

> Node.js®是一个免费、开源、跨平台的JavaScript运行时环境，
> 允许开发人员创建服务器、web应用程序、命令行工具和脚本。

1. 打开官网：<https://nodejs.org>
2. 下载安装程序
3. 根据提示安装

## 检查安装是否成功

1. 按下```Windows徽标```+```X```
2. 点击弹出窗口中的```终端管理员```
3. 输入：```node -v```
4. 如果正常显示版本号则成功，如果不正常就重新安装
5. 输入：```npm -v```
6. 如果正常显示版本号则成功，如果不正常就重新安装
