# 安装环境 | 部署Bitwarden密码服务

## 安装Nginx

1. 安装：```sudo apt install nginx```
2. 测试：使用浏览器打开服务器ip，观察是否显示nginx欢迎页

## 安装Docker

```sudo apt install docker.io```

## 安装Docker-compose

```sudo apt install docker-compose```

## 安装OpenSSL

```sudo apt install openssl```
