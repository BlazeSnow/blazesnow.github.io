# 修复桌面图标

> 此教程用于清除桌面图标缓存，可以修复图标变空白的问题

1. 按下```Windows徽标```+```R```
2. 输入```%appdata%```并回车
3. 此时系统会打开```%appdata%/Roaming```文件夹
4. 返回上一级文件夹
5. 打开```%appdata%/Local```文件夹
6. 找到```IconCache.db```文件
7. 删除```IconCache.db```文件
8. 重新启动文件资源管理器，参见<a href="/Windows相关/重启文件资源管理器" target="_blank">重启文件资源管理器</a>
