# 系统配置

> msconfig即系统配置实用程序，是Microsoft System Configuration的缩写。
> 是在开始菜单里运行中输入然后确认就可以找到程序开启或者禁用，
> 可以帮助电脑禁止不需要运行的程序，这样可以加快你的电脑运行。

1. 按下```Windows徽标```+```R```
2. 输入```msconfig```
3. 回车
4. **警告**：
	- 选择```诊断启动```会导致重新启动后无法解锁系统
	- 关闭```Microsoft相关启动项```会导致不可预知的问题
