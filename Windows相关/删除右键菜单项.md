# 删除右键菜单项

## 桌面空白位置右键菜单项

1. 若需要删除的项目属于应用程序```abc```
2. 按下```Windows徽标```+```R```
3. 输入```regedit```并回车
4. 找到```HKEY_CLASSES_ROOT\Directory\Background\shell```
5. 在此文件夹内找到```abc```
6. 删除```abc```文件夹

::: danger 警告：请勿随意修改注册表
对注册表的随意修改会导致不可预知的问题
:::
