# 配置文件

## Windows客户端

1. 前往解压```frp_版本号_windows_amd64.zip```的目录
2. 修改文件：```frpc.toml```

### ```frpc.toml```

```toml{1,3,6,9,10}
serverAddr = "此处修改为服务器ip地址"
serverPort = 7000
auth.token = "此处设置连接密码"

[[proxies]]
name = "此处设置自定义备注名"
type = "tcp"
localIP = "127.0.0.1"
localPort = 此处修改为本地端口号
remotePort = 此处修改为远程端口号
```

## Ubuntu服务端

1. 前往工作目录：```cd /var/frp```
2. 可执行文件提权：```chmod 700 frps```
3. 修改配置文件：```nano frps.toml```

### ```frps.toml```

```toml{2}
bindPort = 7000
auth.token = "此处设置连接密码"
```
