# 开放端口 | 搭建DNS服务器

前往服务器安全组配置处，添加以下端口：

| 端口类型 | 端口  |   端口用途   |
| :------: | :---: | :----------: |
|   UDP    |  53   |   DNS查询    |
|   TCP    |  53   |   DNS查询    |
|   TCP    |  80   |   管理界面   |
|   TCP    |  443  |   管理界面   |
|   TCP    | 3000  | 初始化服务器 |
