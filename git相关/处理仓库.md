# 处理仓库

> 处理git仓库，需要在目标文件夹内操作

## 初始化git仓库

```git init```

## 查看仓库状态

```git status```

## 查看当前的修改

```git diff```

## 添加文件到暂存区

```git add .```

## 提交暂存区的更改

```git commit -m "message"```

## 将本地提交推送到远程仓库

```git push```

## 从远程仓库拉取并合并到本地

```git pull```

## 查看提交历史

```git log```
