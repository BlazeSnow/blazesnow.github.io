# 安装环境

## 安装Nginx

```sudo apt install nginx```

### 测试nginx

1. 启动Nginx：```sudo systemctl start nginx```
2. 查看Nginx状态：```sudo systemctl status nginx```
3. 输入```wq```退出日志
4. 用浏览器访问服务器IP地址
5. 查看是否显示Nginx初始页面
6. 设置Nginx自启动：```sudo systemctl enable nginx```

## 安装Docker

```sudo apt install docker.io```

## 安装Docker-compose

```sudo apt install docker-compose```
