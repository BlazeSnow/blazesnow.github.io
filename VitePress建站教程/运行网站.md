# 运行网站

1. 在网站文件夹中右键
2. 点击```终端```
3. 输入```npm run docs:dev```
4. 随后点击```终端```中输出内容的```http://localhost:端口号```访问网站
5. 回到```终端```
6. 按下```Ctrl```+```C```结束网站

## 验证网站是否正常

1. 按照```运行网站```的方法
2. 将命令更改为：```npm run docs:build```
3. 构建完毕后输入以预览：```npm run docs:preview```
4. 回到```终端```
5. 按下```Ctrl```+```C```结束网站
