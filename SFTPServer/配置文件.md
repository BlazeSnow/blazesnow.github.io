# 配置文件 | 搭建SFTP文件服务器

1. 前往var目录：```cd /var```
2. 创建工作目录：```mkdir sftp```
3. 进入工作目录：```cd sftp```
4. 创建compose文件：```touch compose.yml```
5. 编辑compose文件：```nano compose.yml```

## ```compose.yml```

```yml{9}
version: '3'
services:
  sftp:
    image: atmoz/sftp
    volumes:
      - ./home:/home
    ports:
      - "2222:22"
    command: foo:123:1001:100:/abc
    # 用户名:密码:用户ID:组ID:用户文件夹名
    # 用户文件夹：./home/foo/abc
```
