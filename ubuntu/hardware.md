# <TitleIcon icon="ubuntu" /> 查看硬件配置

## 查看内存与交换分区大小

```bash
free -h
```

## 查看CPU配置

```bash
lscpu
```

## 查看硬盘分区及使用状况

```bash
df -h
```

## 查看系统版本

```bash
cat /etc/os-release
```

## 查看ip地址

```bash
ip address
```
