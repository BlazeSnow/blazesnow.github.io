# <TitleIcon icon="ubuntu" /> 下载文件

## wget

```shell
# 前往保存文件的目录
cd /home

# 下载文件
wget http://example.com/example.txt
```

## curl

```shell
# 前往保存文件的目录
cd /home

# 下载文件
curl -O http://example.com/example.txt
```
