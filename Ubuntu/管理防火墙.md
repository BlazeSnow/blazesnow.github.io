# 管理防火墙 | Ubuntu相关

## 安装防火墙

```apt install ufw```

## 开启防火墙

```ufw enable```

## 禁用防火墙

```ufw disable```

## 放行80端口

> 将会同时放行ipv4和ipv6的端口

```ufw allow 80```

## 阻断3389端口

> 将会同时阻断ipv4和ipv6的端口

```ufw deny 3389```

## 查看防火墙规则

- 查看：```ufw status```
- 带编号查看：```ufw status numbered```

## 删除防火墙规则

- 按编号删除：```ufw delete 2```
- 按规则删除：```ufw delete allow 80```
- 按规则删除：```ufw delete deny 3389```

## 重载防火墙

> 手动修改ufw的配置文件后，需要重新加载防火墙

```ufw reload```
