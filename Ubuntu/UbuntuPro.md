# 激活UbuntuPro | Ubuntu

## 查看UbuntuPro状态

```bash
sudo pro status
```

## 获取token

1. 前往官网注册账户：<https://ubuntu.com/pro/subscribe>
2. 获取Pro后，复制token备用

## 激活UbuntuPro

```bash
sudo pro attach 此处替换为token
```

## 取消激活UbuntuPro

```bash
sudo pro detach
```
