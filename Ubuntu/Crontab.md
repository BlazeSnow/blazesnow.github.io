# 自动化脚本 | Ubuntu

> 本文针对当前登陆用户的自动化脚本

## 查看自动化脚本

```bash
crontab -l
```

## 编辑自动化脚本

```bash
crontab -e
```

## 自动化脚本举例

```shell
0 2 * * * sh /srv/autoapt.sh
```

## 自动化脚本规则

| 分钟  | 小时  |  日   |  月   | 星期几 |         命令         |
| :---: | :---: | :---: | :---: | :----: | :------------------: |
|   0   |   2   |   *   |   *   |   *    | `sh /srv/autoapt.sh` |
