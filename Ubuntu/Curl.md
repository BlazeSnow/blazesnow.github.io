# 下载文件 | Ubuntu

## wget

1. 获取文件链接
2. 打开Ubuntu终端
3. 打开需保存文件的目录，如：`cd /var`
4. 输入：`wget 文件链接`，如：`wget http://example.com/example.txt`

## curl

1. 安装curl：`sudo apt install curl`
2. 获取文件链接
3. 打开Ubuntu终端
4. 打开需保存文件的目录，如：`cd /var`
5. 输入：`curl -O 文件链接`，如：`curl -O http://example.com/example.txt`
