# 前言 | Ubuntu

> Ubuntu（国际音标：/ʊˈbʊntuː/，uu-BUUN-too）是基于Debian，以桌面应用为主的Linux发行版。Ubuntu有三个正式版本，包括桌面版、服务器版及用于物联网设备和机器人的Core版。前述三个版本既能安装于实体电脑，也能安装于虚拟电脑。从17.10版本开始，Ubuntu以GNOME为默认桌面环境。

官方网站：<https://ubuntu.com/>
