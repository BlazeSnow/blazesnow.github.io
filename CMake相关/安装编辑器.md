# 安装编辑器

## 安装VScode

1. 打开软件官网：<https://code.visualstudio.com>
2. 下载并安装

## 安装Chinese语言

1. 打开```VScode```
2. 按下```Ctrl```+```Shift```+```X```
3. 搜索```Chinese```
4. 安装：```Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code``` ```Microsoft```
5. 根据提示重启```VScode```

## 安装相关插件

1. 打开```VScode```
2. 按下```Ctrl```+```Shift```+```X```
3. 搜索```C```
4. 安装：```C/C++``` ```Microsoft```
5. 搜索```CMake```
6. 安装：```CMake Tools``` ```Microsoft```
