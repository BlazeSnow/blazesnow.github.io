# 安装IO发行版 | Docker

## 安装IO发行版

```bash
sudo apt install docker.io
sudo apt install docker-compose
```

## 验证安装

```bash
docker version
docker-compose version
```

## 卸载IO发行版

```bash
sudo apt purge docker.io
sudo apt purge docker-compose
```
