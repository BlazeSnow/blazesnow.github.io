# 配置文件 | 搭建Webdav文件服务器

1. 前往var目录：```cd /var```
2. 创建工作目录：```mkdir webdav```
3. 进入工作目录：```cd webdav```
4. 创建compose文件：```touch compose.yml```
5. 编辑compose文件：```nano compose.yml```

## ```compose.yml```

```yml{10,11}
version: '3'
services:
  webdav:
    image: bytemark/webdav
    restart: always
    ports:
      - "80:80"
    environment:
      AUTH_TYPE: Digest
      USERNAME: admin
      PASSWORD: 1234
    volumes:
      - ./dav:/var/lib/dav
```
