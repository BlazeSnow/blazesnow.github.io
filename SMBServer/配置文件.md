# 配置文件 | 搭建SMB文件服务器

1. 前往var目录：```cd /var```
2. 创建工作目录：```mkdir samba```
3. 进入工作目录：```cd samba```
4. 创建compose文件：```touch compose.yml```
5. 编辑compose文件：```nano compose.yml```

## ```compose.yml```

```yml{7,8}
services:
  samba:
    image: dockurr/samba
    container_name: samba
    environment:
      NAME: "Data"
      USER: "samba"
      PASS: "secret"
    ports:
      - 445:445
    volumes:
      - ./storage:/storage
```
